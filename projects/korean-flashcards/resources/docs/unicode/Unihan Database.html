Unihan Grid Index

The Unihan repertoire of The Unicode Standard is here divided into blocks of 256 characters each for easy access. Select the particular block you want. Each block is indicated by all but the last two hexadecimal digits of common to all the characters in the block (e.g., "34" for code points U+3400 through U+34FF and "200" for code points U+20000 through U+200FF). (You may also go directly to the data for an individual character by using the text field and "Lookup" button to the left. Enter the code point for the character and click on the button.)

Be aware that the display of individual characters depends on your OS, browser, and available fonts.
